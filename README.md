# CastleTech Portfolio Template v1

A clean, responsive portfolio starter template (hero, projects, skills, contact).

## Live Demo
https://willcastle20.github.io/CastleTech-Portfolio-Template-v1/

## Run Locally
1. Open the folder in VS Code
2. Install **Live Server** extension (if you don’t have it)
3. Right-click `index.html` → **Open with Live Server**
